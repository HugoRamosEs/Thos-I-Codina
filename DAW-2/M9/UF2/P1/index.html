Exercici 3

Punt 1

Cloudinary és un servei que permet gestionar imatges i videos en linia, modificant l'URL, oferint diverses caràcteristiques tals com:
  1. Transformació d'imatges en temps real. Permet escalar, retallar, aplicar filtres a l'imatge.
  2. Formats adaptatius. Permet canviar el format de l'imatge o video, a eleccio de l'usuari.
  3. Optimització d'imatges. Permet optimitzar l'imatge, per minimitzar el tamany i temps de càrrega.
  4. CDN integrat. Utilitza una xarxa de lliurament de contingut (CDN).
  5. API Completa. Ofereix una API que permet als desenvolupadors controlar i gestionar les imatges de manera programàtica, oferint flexibilitat i personalització.

Exercici 5

Punt 1

Thumbor és un servei d'edició d'imatges open-source que permet manipular imatges de manera eficient i flexible. Algunes de les funcionalitats son:
  1. • Caracteristiques bàsiques per a manipular imatges. Aquestes serien retall, rotació, redimensionar... etc.
  2. • Personalització. Permet crear plugins i extensions per adaptar-se a necessitats específiques de l'aplicació.
  3. • Compatibilitat. És compatible amb una àmplia gamma de plataformes, frameworks, llenguatges i entorns de desenvolupament.
  4. • Cache. Emmagatzema en memòria cau les imatges modificades per a un accés més ràpid i eficient en futures crides.
